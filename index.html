Let's Get Started!

Making a Good Company a Great One!

Follow below steps to install "AquaGuide: 1-Minute Reference Guide"?


*****************************************************************

iPHONE USERS:

DOWNLOAD AND INSTALL LATEST VERSION!

  • Tap HERE from your iPhone Or Point Camera to below QR CODE & tap Install





  • If this is your first time installing this app, Continue on to below Trust App

TRUST APP

  • When you first open the app you see a notification that the app isn't trusted on your device. You can dismiss this message.



  • Tap Settings > General > VPN & Device Management > Aqua-Aerobic Systems, Inc.



  • Tap Trust "Aqua-Aerobic Systems, Inc"



  • Tap Trust prompt to verify that you trust the app



  • You are now ready to use the app



  • Open app and Tap Allow Notifications



  • Enjoy!

*****************************************************************

Desktop/Laptop/Tablet/Mobile Phone Users!

We created a web app that runs on all devices and web browsers.

The web app does not have the rich features of the mobile app but can run on all desktops, laptops and mobile browsers!

  • Tap here for the 1-Minute Reference Guide web app!

  • Save to Android Home Screen:
    kebab menu (three vertical dots) > tap "Add to Home Screen button" > tap "Add"

ANDROID USERS:

DOWNLOAD AND INSTALL LATEST VERSION!

  • From an Android Phone, go to Settings > Security (in title) > and check "Unknown Sources" to allow your phone to install apps from sources other than the Google Play Store.
  • Make sure you have fast internet
  • Tap here to install app and accept all prompts (if any is given)
